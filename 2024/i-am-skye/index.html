如果看到这个页面，我想你可能对博客的主题感兴趣。如果你不想听我唠叨的话，可以直接去看看 ZenMind 主题。

我写博客差不多十多年了，最开始在 CSDN、博客园写技术文章，但这两平台不管是写作体验还是阅读体验，真是一言难尽。

接下来，在简书上写了两年，每篇文章的阅读量都还不错，互动的人也挺多，我也很喜欢这个平台，直到它开始搞区块链。另外，最令人难受的，我的多篇文章因为被人投诉抄袭而下架。我也不知道判定抄袭的依据是什么，只给我留了一个申诉的链接，我也懒得申诉，直接删除了平台上的所有文章。

经历这些后，我开始折腾自建博客，期间还自己手撸了一个博客系统，但最终还是回到了 Wordpress，要的功能都有，何必折腾。我比较喜欢简洁的风格，就在 Twenty Twelve 主题上改了改，变成下面这个样子，直到现在我还是非常喜欢。

今年，在腾讯云上购买的服务器快到期了，看了看续费价格，1 核 2G 内存的服务器每年要六百多，再看了看自己如今的更新频率，心里只剩一句不值得。而后，又看了看新服务器以及各个 Wordpress 托管厂商的价格，最具性价比的是 HOSTINGER 提供的 WP 托管服务，当前 (2024 年 3 月底) 的价格是 1150 买 4 年，另外赠送 3 个月，折算下来每月 23 元左右，勉强可以接受。但几年到期后，续费价格得翻两倍。

注意，这里不是打广告，只是单纯的介绍一下我查到的结果，我也不知道它的服务是好是坏，如果你有需要的话，可以自行甄别。

我一直都知道有 Hexo、Hugo 等静态博客方案可以选择，只是舍不得那套主题而已。在免费和付费两种方案之间摇摆了一段时间后，咬了咬牙决定还是用免费的吧，毕竟如今的经济环境也不好，能节约一点是一点吧。

简单把 Hexo 主题页翻了翻，又到 Twitter 上找了找，发现了 ZenMind 主题。这个主题的页面非常干净，首页只有顶部菜单、文章摘要列表、底部版权信息，和我目前使用的大差不差，只需要改改 CSS 就可以实现我现有博客的效果。

最终，在原主题上做了如下修改：

  • 修改首页的 CSS
  • 增加 Font Awesome 图标库 V5 版本 ，用于在页面上展示一些小图标
  • 部分页面修改了一点逻辑，比如，正常主题下，如果菜单项配置成 <i class="fas..."></i> 知识 ，会直接把 HTML 标签展示在页面上，但为了在菜单上显示图标，就改了改主题的源码，但这样有一定的注入风险。

除此之外，再无其它修改，就我这半吊子的前端编码水平，只花了半天时间就修改完成，我想你也可以。基于上面的描述，主题就不开源出来了，况且我也不一定有精力来维护。但如果你实在懒得自己修改，可以联系我，我直接把修改后的主题源码发你。

本来只需要把文章评论打开，大家留言就好。但网站在国内备案的是非交互式网站，是不能有评论的。如有需要，只能麻烦大家单独联系我了。当然，如果你对文章有任何看法，也可以联系我，我会在适当的时候，把大家的反馈以文章的方式公开。

至于联系方式，你在网站上一定能找到的。
