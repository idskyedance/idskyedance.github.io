如果看到这个页面，我想你应该是对本站的主题感兴趣。

博客主题基于 ZenMind 改造：

  • 增加 Font Awesome 图标库 V5 版本，用于在页面上展示一些小图标，网页顶部菜单和底部版权信息都能看到它们的身影
  • 升级 highlightjs 插件到最新版本(11.9)，使用 xcode 的主题，更多主题，可以在 highlight.js官方主题 预览
  • 使用 Rubik 字体，可以在 Google 字体中找到它
  • 其它 CSS 细节调整

最后，ZenMind 主题非常棒，简洁美观。我的改造完全基于个人喜好，已经改得面目全非，更推荐大家使用原主题。
